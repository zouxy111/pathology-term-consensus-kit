# Weak Model Checklist

Before running:

- [ ] Config paths exist.
- [ ] Report columns match exactly.
- [ ] Term catalog has all required fields.
- [ ] Company list is complete.
- [ ] Include/exclude terms are explicit.

After running:

- [ ] `run_manifest.json` status is `pass`.
- [ ] `privacy_report.json` status is `pass`.
- [ ] `scan_log.csv` lists every input file/sheet.
- [ ] `questionnaire.xlsx` has the four fixed director choices.
- [ ] No raw report rows were pasted into the final response.
